# Langflow 사용 가이드

AI Agent Builder로 시작하는 LLM 애플리케이션 개발 가이드

**[langflow-guide.posong.space](https://langflow-guide.posong.space)**

## 소개

폐쇄망(Air-gapped) 환경에서 Langflow를 활용하여 AI Agent를 구축하는 방법을 다루는 한국어/영어/중국어 가이드입니다. 사내 vLLM 기반 API를 OpenAI Compatible 컴포넌트로 연동하는 실습 중심의 내용을 담고 있습니다.
